Logiciel de service Sky-Impulse - INSTALLATION:


Matériel nécessaire pour la mise en service avec PC (paramétrage) : 
 
•   PC / ordinateur portable avec système d'exploitation actuel Windows   
•   Adaptateur de service (USB vers CAN / IXXAT)
•   Câble Ethernet   (L = 5m)
•   Câble test d'alimentation avec fiche européenne
•   Logiciel de service Sky-Impulse 


Configuration requise : 
 
Hardware : 
     RAM >= 4 GB 
     Processeur >= 1 GHz 
     Espace libre sur le disque dur >= 1 GB 
     Interface USB >= 2.0 
 
Système opérateur : 
     Windows 7, 8 / 8.1, 10 
     avec .Net 4.0 complète et plus récent 
	 
	 
ATTENTION !  (Risque d'erreur) 
 
Ne pas encore raccorder l'adaptateur de service au PC ! 
Le logiciel doit tout d'abord être installé. 

3 programmes doivent être installés, pour cela ouvrir le répertoire d'installation Sky-Frame Impulse : 

1.) Selon le système d'exploitation (x86 / x64 bits), exécuter tout d'abord l'installateur du logiciel « SkyImpulse » et suivre les indications s'affichant à l'écran. 
2.) Exécuter ensuite le pilote IXXAT dans le répertoire Ixxat : vci_3_5_2_4072.exe
3.) Exécuter ensuite, en fonction du système d'installation (x86 / x64 bits), l'installateur correspondant 
pour « MSCompact » : SSCERuntime_xyy-ENU.exe
4.) Redémarrer alors le PC qui est alors prêt pour le démarrage initial de Sky-Impulse. 

Pour plus d'informations sur l'installation et la mise en service consulter le manuel d'installation de la page 13.4.1.46 sous la rubrique «Docs».
